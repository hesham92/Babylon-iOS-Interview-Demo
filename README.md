### Requirements

* Xcode 10
* Swift 4.2
* CocoaPods (for dependency management)

### Installation:

Just clone the repository, run pod install and you're all set.

### Dependencies (via CocoaPods):
  * pod 'Kingfisher', '~> 5.0'
  * pod 'R.swift'
  * pod 'NotificationBannerSwift'
  * pod 'RealmSwift'
  * pod 'ReachabilitySwift'
